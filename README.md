# drag-and-drop-js

## Description
This is a simple project, created with HTML, CSS and JavaScript.
It allows user to drag and drop an item to different placeholder
under the three categories: Start, In Progress and Done.

## Technologies
- HTML
- CSS 
- Vanila JavaScript
